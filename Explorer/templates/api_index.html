Supported APIs:

  • /api/addressbalance/ADDRESS - Amount ever received minus amount ever sent for the specific address
  • /api/blockcount/ - The current block number
  • /api/confirmations/BLOCK_HASH .. OR .. /api/confirmations/BLOCK HEIGHT - Total confirmations for the block hash/block height
  • /api/connections/ - The number of connections to other nodes
  • /api/lastdifficulty/ - Last solved block's difficulty
  • /api/mempool/ - View the current mempool
  • /api/peers/- Current peers connected to this instance
  • /api/receivedbyaddress/ADDRESS - Amount ever received by a given address
  • /api/richlist/ - Addresses with the most coins
  • /api/sentbyaddress/ADDRESS - Amount ever sent from a given address
  • /api/totalcoins - Amount of all coins on the network
  • /api/totaltransactions/ - Total number of all transactions
  • /api/validateaddress/ADDRESS - Checks an address for validity.
